# CommentPulse - 评论脉搏

B站评论分析浏览器插件，实时分析视频评论情绪、关键词和精选评论。

## 功能特性

- 📊 **实时情绪分析**：分析评论的积极/中性/消极情绪比例
- ☁️ **关键词词云**：可视化展示评论中的高频关键词
- 💬 **精选评论**：智能筛选三条最需关注的评论
  - 最高赞提问
  - 最有建设性意见
  - 最具代表性的情绪宣泄
- 🤖 **AI回应建议**：为UP主生成30秒回应建议（V1.0使用模拟数据）

## 技术栈

- Vue 3 + TypeScript
- Vite
- Element Plus
- ECharts + ECharts WordCloud
- 基于关键词匹配的中文情感分析（纯 JavaScript 实现）

## 开发

```bash
# 安装依赖
npm install

# 开发模式（监听文件变化）
npm run dev

# 构建
npm run build
```

## 安装

1. 构建项目：`npm run build`
2. 打开 Chrome/Edge 浏览器，进入扩展程序管理页面
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `dist` 目录

## 使用

1. 访问任意B站视频页面
2. 页面右侧会出现一个蓝色图标
3. 点击图标打开分析面板
4. 插件会自动抓取并分析评论

## 注意事项

- 评论抓取需要页面已加载评论区域
- 插件使用Shadow DOM隔离样式，不会影响B站页面
- 情感分析基于关键词匹配，准确度取决于评论内容
